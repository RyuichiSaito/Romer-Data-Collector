# Romer-Data-Collector

・研究用  

MFCをベースに開発した三次元測定機データ収集ソフトです．<br>
RDS APIを用いてCMMからデータを収集します．  

使用したAPI
* RDS API : v0.3.7 

## Requirement
 
* RDS Agent : v5.4.1 (Nikon edition)
* Visual Studio 2019


## Usage

slnファイルをVisual Studioで開いてビルドする.
